\# Credit Card Fraud Detection \& Business Impact Analysis



Analysis of 284,807 anonymized credit card transactions to identify fraud patterns and evaluate detection tradeoffs from a business perspective.



\## Data Source

\[Kaggle: Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud) (not included in repo due to size — download separately)



\## Project Structure

\- `sql/` — exploratory SQL queries

\- `notebooks/` — visualization and modeling

\- `images/` — exported charts
